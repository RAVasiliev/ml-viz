k ближайших соседей (kNN)

Метрический алгоритм без обучения параметров (lazy learning): класс новой точки берётся голосованием (majority vote) её k ближайших соседей из обучающей выборки по выбранной метрике расстояния. Фон закрашен предсказанным классом, насыщенность = доля голосов за победителя. Покрути k: при k=1 граница изрезанная (переобучение на шуме), при большом k — гладкая. Клик по полю показывает «бюллетень» голосов.

k=1 → извилистая граница и переобучение на шуме; большое k → усреднение и гладкая граница.
Метрика задаёт форму «шара» близости: круг (L2), ромб (L1), квадрат (L∞).
Взвешенный kNN: ближним соседям больше веса через ядерную функцию K(d).
