Метод опорных векторов (SVM)

SVM ищет разделяющую гиперплоскость с максимальным отступом. Решающая функция f(x) = Σ αᵢyᵢK(xᵢ,x) построена на обучающих точках; обводятся опорные векторы — те, что нарушают отступ y·f(x) < 1 и держат границу, как колонны крышу. Сплошная линия — граница f=0, пунктир — полоса f=±1. Параметр C — мягкий отступ (баланс ширины и ошибок). С ядерным трюком (RBF) граница становится нелинейной: linear не разделит кольца, а RBF — да. Обучение — Pegasos (стохастический субградиент Hinge Loss).

Кольца: linear не разделит, RBF — да (покрути γ).
Ядерный трюк: ⟨xᵢ,xⱼ⟩ → K(xᵢ,xⱼ). RBF лепит нелинейную границу, linear — только прямую.
Параметр RBF-ядра: больше γ → сложнее граница и риск переобучения.
Штраф за ошибки и нарушение отступа. Больше C → уже полоса и жёстче; меньше C → шире отступ, больше точек внутри (slack ξᵢ).
